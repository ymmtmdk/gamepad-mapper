# GamepadMapper 内部資料

このフォルダには、プロジェクト管理に必要な最小限の内部情報が含まれています。

## 📁 資料一覧

| ファイル | 目的 | 更新頻度 |
|----------|------|---------|
| **[PROJECT_STATUS.md](PROJECT_STATUS.md)** | 現在のプロジェクト状態・品質レベル | 月次 |
| **[IMPROVEMENT_HISTORY.md](IMPROVEMENT_HISTORY.md)** | 主要改善の記録・教訓 | 改善完了時 |

## 📊 使用方法

### プロジェクト状況確認時
- `PROJECT_STATUS.md` で現在の品質・技術レベルを確認
- 次期改善候補・技術負債の把握

### 新しい改善作業時
- `IMPROVEMENT_HISTORY.md` で過去の成功パターンを参考
- 同じ問題の繰り返し回避

### 定期レビュー時
- 品質維持状況の確認
- 技術負債の監視
